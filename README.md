# word-grid-ai
AI-powered Word Search Puzzle Solver using Langflow and Groq Vision. Extracts letter grids from images with Vision AI, converts them into structured 2D matrices, and deterministically solves puzzles across all 8 directions. Outputs accurate, color-highlighted HTML visualizations with reliable indexing and zero hallucinations.

# 🧩 AI Word Grid Solver

An AI-powered word search puzzle solver built using:

- Groq Vision Models
- Langflow Custom Components
- Python

## 🚀 Features
- Extracts letter grid from puzzle image
- Detects words automatically
- Highlights found words
- Beautiful HTML visualization

## 🧠 Tech Stack
- Python
- Langflow
- Groq API (Vision)
- Base64 Image Processing
